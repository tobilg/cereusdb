@cereusdb/standard playground

Query spatial data in the browser.

This playground uses the standard CereusDB browser package: core spatial SQL, GEOS, PROJ, relation joins, distance joins, and ST_KNN. It folds the old static examples into one interface for ad hoc queries, remote Parquet loading, and local file registration.
